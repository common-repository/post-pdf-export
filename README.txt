Do your site has a large number of post where it's too many you find it hard to check or monitor all of it? Here is a plugin that can help you with this kind of concerns. Post PDF Export plugin is a simple plugin that gives instant post download in PDF format. You can apply post download both front-end and back-end of your site with this plugin. With it's great features, you can easily manage the PDF report and display post information the way you want it.This plugin uses DOMPDF library. DOMPDF is an HTML to PDF converter, you can check this library here: http://code.google.com/p/dompdf/== Installation ==1. Upload the files to the /wp-content/plugins/ directory.2. Activate the plugin3. Go to "Post PDF Export" have fun!== Screenshots ==1. Post Export2. Settings3. PDF Template4. Post Download
